Instrucciones 

1. Consultar acerca de los métodos de ordenamiento
2. Se debe evidenciar a través de un video participativo el cual debe ser construido por los estudiantes y presentado a sus compañeros de curso explicando el funcionamiento interno de cada uno de los métodos consultados.
3. Crear un programa usando el lenguaje de su elección, el cual demuestre el correcto funcionamiento de los métodos de ordenamiento.
4. Crear un menú de opciones con a siguiente estructura:
   · Método Burbuja
   · Método Secuencial
   · Método Quicksort
   · Salir: Esta opción permite finalizar la aplicación.

Condiciones de entrega 

   · El ejercicio debe estar codificado en lenguaje de su elección
   · Se debe generar el archivo fuente en cualquier lenguaje de alto nivel
   · Se debe entregar el proyecto generado
   · Se debe realizar la documentación interna del código mediante comentarios en cada una de las partes del código
