Instrucciones 

1. Realizar una investigación acerca de listas circulares dinámicas.
2. Realizar una aplicación usando estructuras lineales circulares.
3. Se debe evidenciar el proceso de creación desde la programación orientada a objetos.
4. Crear un programa usando el lenguaje de programación PYTHON aplicando conceptos de listas circulares circulares.
5. Crear un menú de opciones con a siguiente estructura:

 - Insertar cliente: Esta opción permite pedir la cedula y nombre del cliente a ser insertado sobre la lista simple.
 - Para este ejercicio solo se podrá crear la variable nueva para crear un nuevo nodo y la variable Inicio para identificar la lista. Esta variable se ubicará en el último nodo de la lista una vez que se haya realizado la inserción.
 - Listar Clientes hacia la derecha: Esta opción permitirá imprimir los n clientes que han sido cargados sobre la lista circular desde el primer nodo hasta el último nodo de lista.
 - Salir: Esta opción permite finalizar la aplicación.

Condiciones de entrega 

 - El ejercicio debe estar codificado en lenguaje de programación PYTHON
 - Se debe generar el archivo fuente en un código java
 - Se debe entregar el proyecto generado desde usando el IDE de su elección
 - Se debe realizar la documentación interna del código mediante comentarios en cada una de las partes del código
